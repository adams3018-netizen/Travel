# Courier Route Companion — Ready to Publish

Upload every file and folder in this package to the root of a GitHub repository.

Then enable GitHub Pages:
Settings → Pages → Deploy from a branch → main → /(root)

Share the GitHub Pages URL. On a phone, open the link in Safari or Chrome.
The game can then be added to the home screen.

Do not upload the ZIP itself to the repository. Unzip it first and upload its contents.
